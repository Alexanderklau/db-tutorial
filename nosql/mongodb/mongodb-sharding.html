        MongoDB ТЋ░ТЇ«т║ЊтЈ»С╗ЦтљїТЌХтїЁтљФтѕєуЅЄтњїТюфтѕєуЅЄуџёжЏєтљѕуџё collectionсђѓтѕєуЅЄ collection С╝џтѕєтИЃтюежЏєуЙцСИГтљёУіѓуѓ╣СИісђѓУђїТюфтѕєуЅЄуџё collection тГўтѓетюеСИ╗Уіѓуѓ╣СИісђѓТ»ЈСИфТЋ░ТЇ«т║ЊжЃйТюЅтЁХУЄфти▒уџёСИ╗Уіѓуѓ╣сђѓ

        # тѕєуЅЄ Key

        MongoDB Сй┐ућетѕєуЅЄ Key тюетљёСИфтѕєуЅЄС╣ІжЌ┤тѕєтЈЉ collection уџё documentсђѓтѕєуЅЄ Key ућ▒ document СИГуџёСИђСИфТѕќтцџСИфтГЌТ«ху╗ёТѕљсђѓ

        • С╗ј MongoDB 4.4 т╝ђтДІ№╝їтѕєуЅЄ collection СИГуџё document тЈ»УЃйу╝║т░ЉтѕєуЅЄ Key тГЌТ«хсђѓтюеУиетѕєуЅЄтѕєтИЃТќЄТАБТЌХ№╝їу╝║т░ЉтѕєуЅЄ Key тГЌТ«хт░єУбФУДєСИ║тЁиТюЅуЕ║тђ╝№╝їСйєтюеУи»ућ▒ТЪЦУ»бТЌХтѕЎСИЇС╝џсђѓ

        • тюе MongoDB 4.2 тЈіТЏ┤ТЌЕуЅѕТюгСИГ№╝їтѕєуЅЄ Key тГЌТ«хт┐ЁжА╗тюеТ»ЈСИф document СИГтГўтюеСИђСИфтѕєуЅЄ collectionсђѓ

        тюетѕєуЅЄ collection ТЌХжђЅТІЕтѕєуЅЄ Keyсђѓ

        • С╗ј MongoDB 4.4 т╝ђтДІ№╝їТѓетЈ»С╗ЦжђџУ┐Єтюеуј░ТюЅ Key СИГТи╗тіаСИђСИфТѕќтцџСИфтљју╝ђтГЌТ«хТЮЦС╝ўтїќ collection уџётѕєуЅЄ Keyсђѓ
        • тюе MongoDB 4.2 тњїТЏ┤СйјуЅѕТюгСИГ№╝їТЌаТ│ЋтюетѕєуЅЄтљјТЏ┤Тћ╣тѕєуЅЄ Key уџёжђЅТІЕсђѓ

        document уџётѕєуЅЄжћ«тђ╝тє│т«џС║єтЁХтюетљёСИфтѕєуЅЄСИГуџётѕєтИЃ

        • С╗ј MongoDB 4.2 т╝ђтДІ№╝їжЎцжЮъТѓеуџётѕєуЅЄ Key тГЌТ«хТў»СИЇтЈ»тЈўуџё_id тГЌТ«х№╝їтљдтѕЎТѓетЈ»С╗ЦТЏ┤Тќ░ document уџётѕєуЅЄжћ«тђ╝сђѓ
        • тюе MongoDB 4.0 тЈіТЏ┤СйјуЅѕТюгСИГ№╝їТќЄТАБуџётѕєуЅЄ Key тГЌТ«хтђ╝Тў»СИЇтЈ»тЈўуџёсђѓ

        тѕєуЅЄ Key у┤бт╝Ћ№╝џУдЂт»╣ти▓тАФтЁЁуџё collection У┐ЏУАїтѕєуЅЄ№╝їУ»Ц collection т┐ЁжА╗тЁиТюЅС╗ЦтѕєуЅЄ Key т╝ђтц┤уџёу┤бт╝ЋсђѓтѕєуЅЄСИђСИфуЕ║ collection ТЌХ№╝їтдѓТъюУ»Ц collection У┐ўТ▓АТюЅжњѕт»╣ТїЄт«џтѕєуЅЄ Key уџёжђѓтйЊу┤бт╝Ћ№╝їтѕЎ MongoDB С╝џтѕЏт╗║Тћ»ТїЂу┤бт╝Ћсђѓ

        тѕєуЅЄ Key уГќуЋЦ№╝џтѕєуЅЄ Key уџёжђЅТІЕС╝џтй▒тЊЇтѕєуЅЄжЏєуЙцуџёТђДУЃй№╝їТЋѕујЄтњїтЈ»С╝Иу╝ЕТђДсђѓтѕєуЅЄ Key тЈітЁХтљјтцЄу┤бт╝ЋуџёжђЅТІЕС╣ЪС╝џтй▒тЊЇжЏєуЙцтЈ»С╗ЦСй┐ућеуџётѕєуЅЄуГќуЋЦсђѓ

        MongoDB тѕєтї║т░єТЋ░ТЇ«тѕєуЅЄсђѓТ»ЈСИфтѕєтЮЌжЃйТюЅтЪ║С║јтѕєуЅЄ Key уџёСИіСИІжЎљсђѓ

        СИ║С║єтюеТЋ┤СИфжЏєуЙцСИГуџёТЅђТюЅтѕєуЅЄСИіт«ъуј░тЮЌуџётЮЄтїђтѕєтИЃ№╝їтЮЄУААтЎетюетљјтЈ░У┐љУАї№╝їт╣ХтюетљётѕєуЅЄСИіУ┐ЂуД╗тЮЌсђѓ

        Т│еТёЈ№╝џСй┐ућетЊѕтИїу┤бт╝ЋУДБТъљТЪЦУ»бТЌХ№╝їMongoDB С╝џУЄфтіеУ«Ау«ЌтЊѕтИїтђ╝№╝їт║ћућеуеІт║ЈСИЇжюђУдЂУ«Ау«ЌтЊѕтИїсђѓ

        т░йу«АтѕєуЅЄ Key УїЃтЏ┤тЈ»УЃйТў»РђюТјЦУ┐ЉРђЮуџё№╝їСйєт«ЃС╗гуџётЊѕтИїтђ╝СИЇтцфтЈ»УЃйтюетљїСИђ chunk (opens new window) СИісђѓтЪ║С║ј Hash уџёТЋ░ТЇ«тѕєтЈЉТюЅтіЕС║јТЏ┤тЮЄтїђуџёТЋ░ТЇ«тѕєтИЃ№╝їт░цтЁХТў»тюетѕєуЅЄ Key тЇЋУ░ЃТЏ┤Тћ╣уџёТЋ░ТЇ«жЏєСИГсђѓ

        СйєТў»№╝їHash тѕєуЅЄТёЈтЉ│уЮђт»╣тѕєуЅЄ Key тЂџУїЃтЏ┤ТЪЦУ»бТЌХСИЇтцфтЈ»УЃйжњѕт»╣тЇЋСИфтѕєуЅЄ№╝їС╗јУђїт»╝УЄ┤ТЏ┤тцџуџёжЏєуЙцУїЃтЏ┤тєЁуџёт╣┐ТњГТЊЇСйюсђѓ
